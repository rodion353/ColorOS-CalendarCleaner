# CalendarCleaner — LSPosed Module

Removes Chinese lunar calendar hieroglyphs from the standard Calendar app on ColorOS (OnePlus/OPPO).

## Requirements

- Android 10+
- ColorOS / OxygenOS (OnePlus / OPPO)
- Root (Magisk)
- LSPosed (Zygisk)

## Installation

1. Download APK from [Releases](../../releases)
2. Install APK on your device
3. Open LSPosed → Modules → enable **CalendarCleaner**
4. Set Scope to **Calendar** (`com.coloros.calendar`)
5. Force stop the Calendar app or reboot
